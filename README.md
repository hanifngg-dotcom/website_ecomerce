# ShopQR E-commerce

Website e-commerce statis dengan fitur QRIS, keranjang, wishlist, checkout, riwayat pesanan, dan promo.

## Fitur utama

- Katalog produk dengan filter dan pencarian
- Keranjang belanja dinamis
- Checkout dengan pilihan QRIS, transfer bank, dan COD
- QRIS payment dengan QR code
- Wishlist produk
- Riwayat pesanan pelanggan
- Kode promo diskon
- Responsif untuk desktop dan mobile

## Cara menggunakan

1. Buka `index.html` di browser.
2. Pilih produk dan tambahkan ke keranjang.
3. Buka keranjang, lalu klik `Checkout`.
4. Pilih metode pembayaran `QRIS` untuk melihat QR code.

> Data keranjang, wishlist, dan pesanan disimpan di `localStorage` browser.
